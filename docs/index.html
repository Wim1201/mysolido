Het probleem

Je data staat niet bij jou

Persoonlijke documenten, medische dossiers, foto's en wachtwoorden staan verspreid over servers van Amerikaanse techbedrijven. Je hebt geen controle over wie erbij kan.

Geen eigenaarschap

Google, Microsoft en Apple bepalen de voorwaarden. Zij bezitten de servers waarop jouw leven staat.

Geen inzicht

Je weet niet wie jouw data bekijkt, hoe lang het bewaard wordt, of waarvoor het gebruikt wordt.

Geen controle

De CLOUD Act geeft de Amerikaanse overheid toegang tot data op Amerikaanse servers — ongeacht waar jij woont.

De oplossing

Installeer je eigen datakluis

MySolido draait op jouw pc. Je documenten verlaten nooit je computer, tenzij jij dat wilt. Jij bepaalt wie erbij mag en hoe lang.

  • Lokale opslag — Alles staat op je eigen harde schijf
  • Jij bepaalt toegang — Deel specifieke bestanden met specifieke mensen
  • Open standaard — Gebouwd op W3C Solid, geen vendor lock-in
  • Geen abonnement — Eenmalig downloaden, voor altijd van jou

Technologie

Gebouwd op open standaarden

Geen propriëtaire technologie. MySolido gebruikt het Solid protocol — een open W3C-standaard ontwikkeld door Tim Berners-Lee, de uitvinder van het wereldwijde web.

🔷
Solid Protocol W3C-standaard voor data-soevereiniteit
🌐
W3C Open webstandaarden
⚙️
Community Solid Server Open source pod-opslag
🐍
Flask / Python Gebruikersinterface
Dezelfde standaard als de groten. Athumi (Vlaamse overheid), de Nederlandse Datakluis (Jouw.id) en de EU Digital Identity Wallet bouwen allemaal op Solid. MySolido is volledig interoperabel.
